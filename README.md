# Oxygen

Oxygen is a clean, sidebar based Roblox executor UI.
Built to be simple, fast, and out of the way.

No bloat. No noise. Just tools that work.

---

## Features

* Sidebar based layout
* Themes
* Script manager
* Script hub
* Folder support
* Autostart
* Stay on top
* Minimal and responsive UI

Everything is where it should be.

---

## Requirements

* You know what a Roblox executor is
* You know how to use it
* Bash compatible environment

---

## Install

Run this command:

```bash
curl -fsSL https://raw.githubusercontent.com/Zerite29/Oxygen/refs/heads/main/install.sh | bash
```
